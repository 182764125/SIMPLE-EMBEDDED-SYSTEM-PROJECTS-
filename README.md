# Simple Embedded System Projects
   
Project 1: Smart Lighting System
Working Principle: When the light level is low (detected by LDR), the LED turns ON automatically.
  Components: Arduino Uno, LDR Module, LED, 220 Ohm Resistor.
Application:   Smart Street Lights.

## Project 2: Motion Detection Alarm
Working Principle: If a movement is detected by the PIR sensor, the Buzzer and LED turn ON.
Components: Arduino Uno, PIR Sensor, Buzzer, LED.
   Application:Home Security Systems.

### Project 3: Temperature Monitoring System
Working Principle: Monitors the surrounding temperature using a sensor and displays status. Components: Arduino Uno, Temperature Sensor, LED.
Application:Industrial Safety.
